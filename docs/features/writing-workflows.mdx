# Writing Workflows with Cline Writer

Cline Writer provides specialized workflows for different types of writing projects. This guide covers the main approaches and best practices.

## PLAN Mode vs WRITE Mode

### PLAN Mode
Use PLAN mode when you need to:
- Brainstorm and organize ideas
- Create outlines and structural frameworks
- Conduct research and gather sources
- Develop character profiles or thematic analysis
- Plan narrative arcs or argument structures

**In PLAN mode, Cline Writer will NOT modify your files.** Instead, it focuses on:
- Reading your existing work and notes
- Providing strategic guidance
- Helping organize your thoughts
- Creating comprehensive outlines
- Identifying research needs

### WRITE Mode  
Switch to WRITE mode when you're ready to:
- Draft new content based on your plans
- Revise existing manuscripts
- Expand outline sections into full prose
- Edit and refine your writing
- Format citations and references

**In WRITE mode, Cline Writer can modify your files** and will:
- Create new document sections
- Edit existing content
- Format your writing according to style guidelines
- Insert citations and references
- Restructure content based on your feedback

## Project Organization

### Memory Bank Structure
```
memory_bank/
├── project_state.md      # Current status and goals
├── style_guide.md        # Voice, tone, and formatting preferences
├── research_template.md  # Research notes and sources
└── outlines/
    ├── academic_paper_template.md
    ├── novel_template.md
    └── screenplay_template.md
```

### Typical Project Layout
```
your-writing-project/
├── memory_bank/          # Project memory and templates
├── drafts/              # Working manuscripts
├── research/            # Source materials and notes
├── outlines/            # Structural planning documents
└── final/               # Completed sections
```

## Academic Writing Workflow

1. **Research Phase (PLAN mode)**
   - Use research templates to organize sources
   - Create literature review outlines
   - Develop thesis statements and arguments

2. **Outline Phase (PLAN mode)**
   - Structure your paper using academic templates
   - Plan section headers and argument flow
   - Identify evidence needs for each section

3. **Drafting Phase (WRITE mode)**
   - Expand outline sections into full prose
   - Insert citations and references
   - Maintain academic voice and style

4. **Revision Phase (WRITE mode)**
   - Refine arguments and evidence
   - Improve clarity and flow
   - Format according to style guidelines

## Creative Writing Workflow

### Novel Writing
1. **Character Development (PLAN mode)**
   - Create character profiles and arcs
   - Plan relationships and conflicts
   - Develop backstories and motivations

2. **Plot Structure (PLAN mode)**
   - Use three-act structure or other frameworks
   - Plan major plot points and scenes
   - Develop subplots and themes

3. **Chapter Planning (PLAN mode)**
   - Break story into manageable sections
   - Plan POV and narrative structure
   - Set scene goals and conflicts

4. **Drafting (WRITE mode)**
   - Write scenes based on outlines
   - Develop dialogue and description
   - Maintain narrative voice consistency

### Screenplay Writing
1. **Concept Development (PLAN mode)**
   - Develop loglines and premises
   - Plan three-act structure
   - Create character arcs

2. **Treatment Writing (PLAN mode)**
   - Expand story into detailed treatment
   - Plan major scenes and sequences
   - Develop visual storytelling elements

3. **Script Formatting (WRITE mode)**
   - Convert treatment to proper screenplay format
   - Write scene descriptions and dialogue
   - Format according to industry standards

## Research Integration

### Source Management
- Use the research template for consistent note-taking
- Organize sources by topic or relevance
- Track citations for easy reference

### Fact Checking
- Verify important details during writing
- Cross-reference multiple sources
- Note areas requiring additional research

### Citation Integration
- Include citations as you write
- Use consistent formatting style
- Maintain bibliography as you progress

## Collaboration Features

### Version Control
- Use clear file naming conventions
- Keep draft versions for comparison
- Track major revisions in project state

### Feedback Integration
- Document feedback in memory bank
- Plan revisions based on notes
- Track changes and improvements

## Tips for Success

1. **Start in PLAN mode** - Always begin with planning before jumping into writing
2. **Use the memory bank** - Keep project information organized and accessible  
3. **Break work into sessions** - Focus on specific goals each time
4. **Switch modes appropriately** - Plan when organizing, write when creating
5. **Update project state** - Keep memory bank current with your progress

## Common Commands

- Switch to PLAN mode: "Let's switch to PLAN mode to work on the outline"
- Switch to WRITE mode: "Switch to WRITE mode so we can start drafting"
- Check project status: "What's the current state of this project?"
- Generate outline: "Create an outline for [project type]"
- Research assistance: "Help me research [topic]"

This workflow system helps ensure your writing projects stay organized, well-researched, and consistently executed from conception to completion.
