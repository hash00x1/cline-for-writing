# Writing Project Template

This folder serves as the central repository for all project materials and state information. Cline Writer uses this folder to maintain continuity across sessions and understand the current project context.

## Folder Structure

### `/outlines/`
Contains project outlines, structure plans, and organizational documents:
- Main project outline
- Chapter/section breakdowns
- Story beats (for creative writing)
- Thesis structure (for academic writing)

### `/research/`
Research materials, notes, and reference documents:
- Literature review notes
- Source summaries
- Research questions
- Background materials
- Data and statistics

### `/drafts/`
Working drafts and versions:
- Current draft files
- Previous versions
- Section drafts
- Experimental versions

### `/characters/` (Fiction projects)
Character development materials:
- Character profiles
- Character arcs
- Dialogue samples
- Relationships mapping

### `/citations/`
Bibliography and citation management:
- Reference lists
- Citation database
- Source notes
- Bibliography formatting

### Root Files
- `project_state.md` - Current project status and next steps
- `style_guide.md` - Writing style preferences and guidelines
- `project_notes.md` - General project notes and brainstorming

## Usage Instructions

1. **Starting a New Project**: Create relevant subfolders based on your project type
2. **Daily Writing**: Update project_state.md with progress and next steps
3. **Research Phase**: Store all research materials in the research folder
4. **Drafting Phase**: Keep working drafts organized by version or section
5. **Revision Phase**: Maintain version history and track changes

## Best Practices

- Keep filenames clear and descriptive
- Use consistent naming conventions
- Update project_state.md regularly
- Organize research by topic or source
- Back up important drafts regularly
